Explicaciones:

***Ejercicio7TextView:
Este ejercicio contiene un Listview en el cual se podrá ver una lista de países
al dar clic sobre uno de los países se podrá ver en el TextView los habitantes
referentes a ese país el arreglo habitante no contiene datos reales.
***Ejercicio11:
Contiene un TexView que contiene un mensaje de lo que el botón puede hacer
Al dar clic en el botón nombre este desencadenara una acción para mostrar una
segunda pantalla en la cual se verá el nombre del programador.
La segunda pantalla contiene también un Textview que muestra el nombre del 
programador y un botón para volver a la primera pantalla.
***Ejercicio13
Contiene un textView el cual se le puede escribir cualquier cosa para 
guardar en un archivo con ayuda de la función asíncrona SharedPreferences.
También contiene 2 botones el cual el primero es para guardar el dato y el 
segundo se usa para retornar los valores.
***Ejerccio30_Dibujar
En este ejercicio contiene una pantalla en blanco que al dar clic sostenido con 
el mouse se podrá dibujar una línea de color azul.
***Ejercicio45
En este ejercicio se presenta un menú desplegable que está oculto con la barra 
superior lado derecho y se activa dando clic en los 3 puntos.
Al activar el menú desplegable se tiene 3 opciones las cuales al dar un clic 
sobre ellas mostrara en la parte inferior de la pantalla la opción que se 
presiono.
